# EHR Annotation Platform - Frontend

An enterprise-grade clinical document review and annotation dashboard. Built with React, Vite, and Ant Design for high-performance medical workflows.

## 🔗 Repository Links
- **Frontend**: [https://github.com/harsh-vardhhan/EHR-frontend](https://github.com/harsh-vardhhan/EHR-frontend)
- **Backend**: [https://github.com/harsh-vardhhan/EHR-backend](https://github.com/harsh-vardhhan/EHR-backend)

## ✨ Key Features

- **Modern Clinical Dashboard**: A professional three-column layout designed for maximum medical data visibility.
- **System-Aware Dark Mode**: Optimized for low-light clinical environments with high-contrast medical highlights.
- **Medical Entity Recognition**: Seamlessly review AI-suggested annotations (Conditions, Medications, Symptoms, Procedures).
- **Enterprise Aesthetics**: Powered by **Ant Design v6**, featuring smooth transitions, consistent spacing, and premium medical typography.
- **Mobile Responsive**: Fully accessible on mobile and tablets with a specialized clinical drawer for metrics.
- **Real-time Metrics**: Dynamic session analysis, accuracy tracking, and label distribution charts.

## 🛠 Tech Stack

- **Framework**: React 19 (TypeScript)
- **Build Tool**: Vite
- **UI Library**: Ant Design v6
- **Icons**: Ant Design Icons
- **Deployment**: AWS S3 & CloudFront (Static Website Hosting)

## 🚀 Getting Started

### Prerequisites
- Node.js 20+
- npm

### Installation
```bash
$ npm install
```

### Development
```bash
$ npm run dev
```

### Build for Production
```bash
$ npm run build
```

## 📦 Deployment
The application is automatically deployed to AWS S3/CloudFront via GitHub Actions upon pushing to the `main` branch.
